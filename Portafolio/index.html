Presupuesto

Este proyecto consiste en una aplicación web diseñada para ayudar a los usuarios a gestionar sus ingresos y gastos de manera eficiente. Utilizando las mismas tecnologías de Front-End (HTML, CSS y JavaScript), hemos creado una herramienta interactiva que permite a los usuarios registrar sus ingresos y egresos, calcular el saldo restante y mostrar el porcentaje de gastos en relación con sus ingresos totales.
